# 在线课堂管理系统设计与实现 - 系统设计文档

## 一、系统概述

本系统是一个基于微信小程序的在线课堂管理平台，主要用于支持教师开展在线教学和学生进行在线学习，实现课程管理、学习资源发布、作业布置与提交、在线签到、成绩统计等功能，提高教学管理效率和学生学习体验。

## 二、角色设计

### 2.1 学生
- **权限**：加入课程、查看学习资源、提交作业、查看成绩
- **功能**：
  - 浏览课程列表和课程详情
  - 加入课程（通过课程码或课程搜索）
  - 查看课程学习资源（文档、课件）
  - 在线签到
  - 查看作业列表和作业详情
  - 提交作业（文件上传、文本提交）
  - 查看作业成绩和评语
  - 查看个人学习进度和成绩统计
  - 查看教师发布的视频
  - 完成教师发布的试卷

### 2.2 教师
- **权限**：创建课程、管理课程、发布资源、布置作业、批改作业、管理签到、查看统计
- **功能**：
  - 创建课程（设置课程名称、课程码、课程简介）
  - 管理课程信息（编辑课程信息、删除课程）
  - 管理课程学生（查看学生列表、移除学生）
  - 发布学习资源（上传文档、课件）
  - 管理学习资源（编辑、删除资源）
  - 创建作业（设置作业标题、内容、截止时间、分值）
  - 查看学生作业提交情况
  - 批改作业（评分、评语）
  - 创建签到活动
  - 手动截止签到
  - 查看签到统计
  - 查看课程数据统计（学生参与度、作业完成率、成绩分布）
  - 进行考试管理（发布试卷，题目管理）
  - 发布教学视频
  - 对学生学习状况的监控
  - 根据学生学习状况进行学习资源的推荐

### 2.3 超级管理员（系统管理员）
- **权限**：账号管理、系统配置、数据管理
- **功能**：
  - 用户账号管理（添加、编辑、删除用户）
  - 角色权限分配（学生/教师）
  - 账号状态管理（启用/禁用）
  - 重置用户密码
  - 查看所有用户列表
  - 查看系统数据统计
  - 系统参数配置

## 三、核心功能模块

### 3.1 用户管理模块
- 用户登录/注册
- 个人信息管理
- **账号管理（仅超级管理员）**：
  - 添加新用户账号
  - 编辑用户信息
  - 删除用户账号
  - 分配用户角色（学生/教师）
  - 启用/禁用账号
  - 重置用户密码
  - 查看用户列表

### 3.2 课程管理模块
- **课程创建与管理**：
  - 创建课程（课程名称、课程码、课程简介、封面图片）
  - 编辑课程信息
  - 删除课程
  - 课程分类管理
  
- **课程加入**：
  - 通过课程码加入课程
  - 搜索课程并申请加入
  - 查看已加入的课程列表

- **课程成员管理**：
  - 查看课程学生列表
  - 移除学生
  - 查看学生基本信息

### 3.3 学习资源模块
- **资源发布**：
  - 上传文档资源（PDF、Word、PPT等）
  - 上传课件资源
  - 设置资源标题、描述、分类
  
- **资源管理**：
  - 编辑资源信息
  - 删除资源
  - 资源分类管理（文档、课件）
  
- **资源学习**：
  - 在线查看文档
  - 下载资源
  - 记录学习进度

### 3.4 作业管理模块
- **作业创建**：
  - 设置作业标题、内容、要求
  - 设置作业截止时间
  - 设置作业分值
  - 上传作业附件（可选）
  
- **作业提交**：
  - 查看作业详情和要求
  - 提交作业（文本内容、文件上传）
  - 修改已提交的作业（截止时间前）
  - 查看提交状态
  
- **作业批改**：
  - 查看学生作业提交列表
  - 查看学生作业内容
  - 评分（设置分数）
  - 填写评语
  - 批量批改
  
- **作业查询**：
  - 查看作业列表（已发布/已截止）
  - 查看作业详情
  - 查看作业成绩和评语

### 3.5 签到管理模块
- **签到创建**：
  - 创建签到活动
  - 教师手动截止签到
  
- **签到参与**：
  - 查看签到活动列表
  - 参与签到（点击签到按钮）
  - 查看签到记录
  
- **签到统计**：
  - 查看签到参与情况
  - 查看签到统计（出勤率、缺勤名单）
  - 导出签到数据

### 3.6 成绩统计模块
- **成绩查看（学生）**：
  - 查看个人作业成绩
  - 查看课程总成绩
  - 查看成绩排名（可选）
  
- **成绩统计（教师）**：
  - 查看课程成绩分布
  - 查看学生成绩列表
  - 导出成绩数据
  - 成绩统计分析

### 3.7 数据统计模块
- **学习数据统计**：
  - 课程学习进度统计
  - 资源学习情况统计
  - 作业完成率统计
  - 签到出勤率统计
  
- **课程数据统计**：
  - 课程参与人数统计
  - 课程活跃度统计
  - 学生参与度分析

## 四、业务流程设计

### 4.1 课程创建与加入流程
```
教师登录 → 创建课程（设置课程信息、生成课程码）→ 学生登录 → 输入课程码或搜索课程 → 申请加入课程 → 加入成功 → 进入课程学习
```

### 4.2 学习资源发布与学习流程
```
教师登录 → 进入课程 → 发布学习资源（上传资源、设置信息）→ 学生登录 → 进入课程 → 查看学习资源 → 在线学习 → 记录学习进度
```

### 4.3 作业布置与提交流程
```
教师登录 → 创建作业（设置作业信息、截止时间）→ 学生登录 → 查看作业 → 提交作业（上传文件/文本）→ 教师查看提交情况 → 批改作业（评分、评语）→ 学生查看成绩
```

### 4.4 签到流程
```
教师登录 → 创建签到活动 → 学生登录 → 查看签到活动 → 点击签到按钮 → 系统记录签到 → 教师手动截止签到 → 教师查看签到统计
```


## 五、技术架构

### 5.1 前端技术栈
- **框架**：Vue.js + uni-app
- **应用端**：微信小程序（基于uni-app开发，学生、教师和管理员均通过小程序操作）
- **UI组件库**：uni-ui
- **状态管理**：Vuex

### 5.2 后端技术栈
- **框架**：Spring Boot
- **数据库**：MySQL 8.0
- **ORM框架**：MyBatis
- **安全框架**：JWT Token
- **文件存储**：本地存储或OSS（阿里云对象存储）

**技术选型**：
   - 前端：Vue.js + uni-app
   - 后端：Spring Boot
   - 数据库：MySQL
   - 开发工具：IDEA

## 六、关键功能点

### 7.1 课程码机制
- 每个课程自动生成唯一课程码
- 学生通过输入课程码快速加入课程
- 课程码可重置（教师）

### 7.2 作业提交机制
- 支持文本和文件两种提交方式
- 文件大小限制（如50MB）
- 支持文件类型限制（如PDF、Word、图片等，禁止视频文件）
- 截止时间前可修改提交内容
- 截止时间后不可提交或修改

### 7.3 签到机制
- **普通签到**：点击签到按钮完成签到
- 教师手动截止签到，截止后学生无法继续签到
- 防止重复签到

### 7.4 学习进度记录
- 记录资源学习状态（未学习/学习中/已完成）
- 统计课程整体学习进度

### 7.5 成绩计算
- 支持作业权重设置
- 自动计算课程总成绩
- 支持成绩导出功能

### 7.6 文件存储
- 支持文件上传（作业、资源）
- 文件大小和类型限制（禁止视频文件上传）
- 文件下载功能
- 文件管理（删除、查看）
